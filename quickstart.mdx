This guide walks you through installing FieldStation42 and launching the web console for the first time. By the end you will have the server running locally and be ready to create your first channel.

## Prerequisites

Before you begin, make sure you have the following:

- A machine running Linux (Raspberry Pi OS Bookworm, Ubuntu, Mint, Debian, or WSL on Windows)
- Python 3.10 or newer
- MPV media player
- `git` installed and available on your `PATH`

## Installation steps

<Steps>
  <Step title="Install system packages">
    Install MPV, Python, and the venv module using your system package manager:

    ```bash
    sudo apt-get install mpv python3 python3-pip python3-venv
    ```
  </Step>
  <Step title="Clone the repository">
    Clone FieldStation42 from GitHub and enter the project directory:

    ```bash
    git clone https://github.com/shane-mason/FieldStation42
    cd FieldStation42
    ```
  </Step>
  <Step title="Run the installer">
    The installer creates a Python virtual environment, installs all dependencies, and sets up the required runtime directories:

    ```bash
    bash install.sh
    ```

    Press **Enter** when prompted to proceed. The installer will not overwrite any existing configuration files.
  </Step>
  <Step title="Activate the virtual environment">
    Activate the Python virtual environment before running any FieldStation42 commands:

    ```bash
    source env/bin/activate
    ```

    You will need to run this command each time you open a new terminal session.
  </Step>
  <Step title="Start the web console">
    Start the FieldStation42 web server:

    ```bash
    python3 station_42.py
    ```

    Running `station_42.py` with no arguments starts the web API server automatically.
  </Step>
  <Step title="Open the web console">
    Open a browser and navigate to:

    ```
    http://localhost:4242
    ```

    The web console lets you create and manage channels, rebuild catalogs, generate schedules, and monitor playback status — all without using the command line.
  </Step>
</Steps>

<Note>
  The web console is also accessible from other devices on your local network. Use your machine's IP address in place of `localhost` to reach it from a phone or another computer.
</Note>
