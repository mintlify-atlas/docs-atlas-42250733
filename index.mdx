FieldStation42 transforms your Linux machine or Raspberry Pi into a fully functioning broadcast and cable TV simulator. Instead of choosing what to watch, you flip channels — and scheduled programming plays on its own timeline, complete with commercials, station bumps, and all the in-between content that made old-school TV feel alive.

## What FieldStation42 does

FieldStation42 manages your media library as a real TV schedule. You define what plays on each channel and when — morning cartoons, prime time dramas, late-night movies — and the system builds a continuous schedule automatically. Every channel flips to the right show at the right time, with commercials filling the breaks.

## Get started

<Steps>
  <Step title="Install dependencies">
    Install Python 3.10+, MPV, and the required system packages on your Linux machine or Raspberry Pi.
  </Step>
  <Step title="Clone and install">
    Clone the repository and run `bash install.sh` to set up the Python virtual environment and dependencies.
  </Step>
  <Step title="Start the web console">
    Activate the environment and run `python3 station_42.py` to start the web server at `http://localhost:4242`.
  </Step>
  <Step title="Create your first channel">
    Open the web console, create a channel, point it at your media directory, and build the schedule.
  </Step>
</Steps>
